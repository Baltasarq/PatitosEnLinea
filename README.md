# Patitos en línea

Una apasionante viaje por la programación por contrato. El principal objetivo de este proyecto es aclarar y ejemplificar el uso de programación por contrato, utilizando C# como lenguaje de programación.

## Objetivo del programa

Una mamá pato lleva a sus patitos en línea. De vez en cuando, tiene que revisar que todos los patitos se dirigen en la misma dirección. Para ello, se utiliza un valor entero en el rango [0, 360), que indica los grados hacia los que cada pato está mirando. Así, la clase **Pato** proporciona *Dir*, la citada dirección, y las clases **MamaPato** y **Patito** derivan de ella.

# Construcción

Se trata de una aplicación de C# estándar, con Avalonia como GUI o interfaz de usuario gráfica. Una vez instalador .NET, el uso de la herramienta `dotnet` estándar creará y ejecutará el objetivo.

    $ dotnet run
